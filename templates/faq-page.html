FAQ'S

Как загрузить данные для анализа?

Данные для анализа необходимо загрузить на страницу загрузки файлов, файл скачивается в формате json за любой период и с любым фильтром (Соцмедиа, СМИ и др) из темы БА:

Что анализирует вкладка Тональный ландшафт / Tonality Landscape

Вкладка Тональный ландшафт анализирует отношение авторов сообщений/публикаций к теме, показывает соотношение позитива к негативу, а также их распределение с возможностью «проваливания» до конкретного автора

Что анализирует вкладка Информационый граф / Information Graph

Информационный граф анализирует распространение информации по теме, в т.ч. с указанием уточняющего поискового запроса, показывает динамику во времени за заданный период с возможностью просмотра автора, источника и аудитории. Возможно задать фильтрацию по типу сообщений: пост, репост, СМИ

Что анализирует вкладка Медиа Рейтинг / Media Rating

Медиа Рейтинг анализирует отношение СМИ к теме мониторинга, в т.ч. с возможностью выбора периода / даты, отслеживает динамику тональности с переходом на текст сообщения на источнике в режиме one-click (по клику)

Что анализирует вкладка Голос Клиента / Client Voice

Вкладка Голос Клиента показывает отношение пользователей к продуктам компании / персоне / бренду, можно задать поисковый запрос, отфильтровать тему по дате, получить разбивку обсуждений на типы сообщений: пост, репост, комментарий, в т.ч. в разрезе их тональности

Что анализирует вкладка Конкуренты / Competititve Environment

Вкладка Конкуренты сравнивает тему с конкурентами и показывает распределение по динамике упоминаний, сравнивает с конкурентами в разрезе по источникам, охвату и тональности

Что анализирует вкладка Внешние факторы / External Factors

Вкладка Внешние факторы подгружает внешние данные (погода, курсы валют, акций и т.д) и служит для оценки влияния и/или корреляции внешних факторов на тему

Страница Данные / Show Data

Вкладка Данные показывает загруженные для анализа темы, а также доп.обработанные файлы тем для их дальнейшей загрузки Projector-кластеризацию

Что анализирует вкладка Кластеризация авторов / Authors Clusterisation

Вкладка Кластеризация авторов показывает «близость» авторов в теме – авторы группируются в кластеры на основе анализа их текстов (близкие по смыслу авторы буду находиться в одном кластере), применяется два подхода:

  • TSNE-кластеризация (для небольшого кол-ва авторов ~до 5 тыс). Рекомендуется для оперативного контроля ситуации (день-два-неделя), когда в явном виде фиксируются одинаковые по высказываниям/публикациям авторы
  • Projector-кластеризация (неограниченное кол-во авторов). Для глубокого анализа за длительный период, когда авторы могут менять свое мнение - оказываться «близкими» уже с другими авторами

Что анализирует вкладка Классификация текстов / Texts Classification

Вкладка Классификация текстов помогает распределить весь поток сообщений по теме на отдельные смысловые тематики, применяется три подхода:

  • Автоматическая разбивка на тематики (BERT/Google алгоритм)
  • Автоматическая группировка на тематики (LDA-алгоритм)
  • Ручная классификация (ручной подбор кол-ва тематик)
